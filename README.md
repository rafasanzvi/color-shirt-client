## Client URLs

| URI path      	| Description                                    	
|---------------	|------------------------------------------------	
| GET         	| `/` 	| return home 	| 
| GET         	| `/shirts` 	| returns gallery shirt 	| 
| GET         	| `/shirts/:id` 	| returns shirt details	| 
| PUT         	| `/shirts/:id/edit` 	|edits shirts 	| 
| POST         	| `/register` 	| registers user 	| 
| POST         	| `/login` 	|checks credentials	| 
| GET           | `/aboutUs` | return about us view |
| GET           | `/myProfile` | return the user profile view |
| GET         	| `/users/list` 	| returns user list 	| 
| GET         	| `/users/:id` 	| returns user details	| 
| PUT         	| `/users/:id/edit` 	| edits user 	| 
| GET           | `/subscribe` | return subscription form |
